**Counter Project**

This is a simple and interactive Counter application built using vanilla HTML, CSS, and JavaScript. It allows users to increase, decrease or reset a numeric value with button clicks.

The Counter project provides a clean and minimal interface to manipulate a number dynamically.

Reset: Resets the counter back to zero.

Real-Time Update: Changes are instantly reflected on the screen.

The project is built using basic DOM manipulation and event handling:

Event Listeners: Each button listens for click events.

State Update: The counter value is updated and rendered dynamically in the UI.
